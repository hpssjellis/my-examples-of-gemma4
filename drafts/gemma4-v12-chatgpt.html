This page runs a large language model entirely in your browser — no server, no internet needed after the first download. The model is stored in your browser cache and loads instantly on future visits, even offline. Uses Gemma 4 E2B-IT (ONNX) via Transformers.js v4.

📖 What is a token? What is a context window?

A token is roughly 3–4 characters or about ¾ of an English word.
"Hello world" ≈ 2–3 tokens. A paragraph ≈ 100–150 tokens.

Input tokens = your message + all previous conversation history sent to the model each time.
Output tokens = the new words Gemma generates in reply. "Max output tokens" caps this.

Context window = total tokens (input + output) the model can hold at once. Gemma 4 E2B = 128,000 tokens. As your conversation grows, old messages eventually fall off.

tok/s = tokens per second. WebGPU + q4 → expect 10–40 tok/s depending on GPU. WASM/CPU → expect 1–5 tok/s.

📷 Camera — capture an image to send with your message

Gemma 4 E2B is multimodal — it can look at images alongside your text.

How it works: the browser draws a video frame onto a hidden Canvas element, converts it to a base64 JPEG data URI, and includes it as an image content block alongside your text prompt. No image is sent to any server — everything stays in your browser.

Note: image support in Transformers.js v4 with the ONNX pipeline may require the model's vision encoder to be enabled. If you see an error, the text-only path always works.



🎤 Microphone — speak your message (Chrome on-device speech recognition)

Uses the browser's built-in Web Speech API (webkitSpeechRecognition) to transcribe your voice to text, then places the transcript in the input box.

In Chrome 139+, recognition uses SODA (Speech On-Device API) — a local model, no audio leaves your machine. Older Chrome sends audio to Google's servers. Firefox and Safari do not support this API.

This is separate from Whisper. A fully offline Whisper pipeline via Transformers.js (~150 MB) is a planned future addition to this demo. Both approaches produce text that Gemma then responds to — Gemma itself does not process raw audio in the ONNX version.
